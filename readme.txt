Abonnement360
(Gestion des clients et abonnements de forfaits télécom)
Objectif du projet :
Le projet Abonnement360 a pour objectif de créer une application web permettant de gérer efficacement les clients et leurs abonnements aux différents forfaits téléphoniques proposés par l’entreprise. L’application comporte deux interfaces distinctes selon le type d’utilisateur : le superutilisateur (admin) et le client.
1- Pour le Superutilisateur (Admin) :
   - Accès à un dashboard complet affichant les statistiques générales et les contenus essentiels.
   - Possibilité de visualiser, ajouter, modifier et supprimer les forfaits disponibles.
   - Consultation de la liste des clients inscrits et des abonnements réalisés par chacun.
   - Accès réservé uniquement à l’admin ; il ne peut pas accéder à l’interface client.
2. Pour le Client :
   - Inscription sur la plateforme via une page Register.
   - Achat de forfaits téléphoniques selon ses besoins.
   - Consultation de ses abonnements et factures.
   - Modification de son profil, y compris la photo et les informations personnelles.
   - Accès réservé uniquement aux clients ; ils ne peuvent pas accéder à l’interface admin.
3. Gestion des accès et sécurité :
   - Une interface de login unique pour tous les utilisateurs avec redirection automatique selon le type d’utilisateur (admin ou client).
   - Contrôle strict des permissions pour garantir que chaque utilisateur n’accède qu’aux fonctionnalités qui lui sont destinées.
